üß™ Exemple complet avec SQLite + Pandas
Supposons que tu as un Excel users.xlsx avec une feuille qui contient :

| name  | email                                         |
| ----- | --------------------------------------------- |
| Alice | [alice@example.com](mailto:alice@example.com) |

import pandas as pd
import sqlite3

# 1. Charger l'Excel
df = pd.read_excel('users.xlsx')  # ou pd.read_excel('users.xlsx', sheet_name='Feuille1')

# 2. Connexion √† la base SQLite (ou autre moteur)
conn = sqlite3.connect('users_db.sqlite')
cursor = conn.cursor()

# 3. Cr√©er la table si elle n'existe pas
cursor.execute("""
    CREATE TABLE IF NOT EXISTS users (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        name TEXT,
        email TEXT
    )
""")

# 4. Ins√©rer les lignes du DataFrame
for _, row in df.iterrows():
    cursor.execute("INSERT INTO users (name, email) VALUES (?, ?)", (row['name'], row['email']))

conn.commit()
conn.close()
